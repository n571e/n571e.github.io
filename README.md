# n571e.github.io

个人主页 | Personal Homepage

## 简介

这是一个简单的个人主页，用于展示个人信息和论文发表情况。

## 如何自定义

打开 `index.html` 文件，修改以下内容：

1. **个人信息**：
   - 姓名、职位、研究方向
   - 单位、职称
   - 邮箱、GitHub 链接

2. **论文列表**：
   - 论文标题
   - 作者列表
   - 发表会议/期刊
   - 相关链接（PDF、代码、幻灯片等）

3. **头像**：
   - 可以将 `<div class="profile-image">👤</div>` 替换为 `<img src="your-photo.jpg" class="profile-image">`
   - 然后添加对应的图片文件

## 部署

将代码推送到 GitHub 后，该主页将自动部署到 `https://n571e.github.io`

## 功能特点

- ✅ 响应式设计，支持移动设备
- ✅ 双语界面（中文/英文）
- ✅ 简洁专业的设计
- ✅ 易于自定义和维护
